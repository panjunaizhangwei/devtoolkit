# DevToolkit 🛠️

Free online developer tools. Fast, privacy-friendly, and open-source.

## Tools

- ✅ JSON Formatter — Format, validate and minify JSON
- ✅ Password Generator — Cryptographically secure random passwords
- 🔜 SQL Beautifier — Format SQL queries
- 🔜 Image Converter — WebP ↔ PNG ↔ JPG
- 🔜 Unit Converter — Length, weight, temperature, etc.
- 🔜 Color Picker — HEX, RGB, HSL converter
- 🔜 Markdown Editor — Live preview
- 🔜 Regex Tester — Test regular expressions
- 🔜 QR Code Generator — Create QR codes
- 🔜 QR Code Reader — Decode QR from images

## Tech Stack

- [Next.js 15](https://nextjs.org/) (App Router)
- [TypeScript](https://www.typescriptlang.org/)
- [Tailwind CSS](https://tailwindcss.com/)
- Deployed on [Vercel](https://vercel.com/)

## Getting Started

```bash
# Install dependencies
npm install

# Run dev server
npm run dev

# Build for production
npm run build
```

Open [http://localhost:3000](http://localhost:3000) in your browser.
